A variable is a named container used to store data in a program. You can think of it like a labeled box where you keep a value, and you can change that value whenever needed.

Simple Explanation
Variable = Name + Value
The value can be numbers, text, or other data.
Example (Python)
x = 10
name = "Partha"
x is a variable storing the number 10
name is a variable storing the text "Partha"
Key Points
Variables store data
They have a name (identifier)
Their value can change during program execution
Real-Life Example

Imagine a box labeled "age":

Today: age = 20
Next year: age = 21

The label stays the same, but the value changes.

A pointer is a special type of variable that stores the memory address of another variable, instead of storing the actual value.

Simple Idea
Normal variable → stores value
Pointer → stores address of value
Example (C Language)
int x = 10;
int *p = &x;
Explanation
x = 10 → normal variable
&x → gives the address of x
p → pointer that stores the address of x
Visual Understanding
x = 10        (stored at address 1000)
p = 1000      (points to x)
Important Symbols
& → Address of variable
* → Value at address (dereference)

Example:

printf("%d", *p);  // prints 10
Key Points
Pointer stores memory address
Helps in efficient memory usage
Used in arrays, functions, dynamic memory
Real-Life Example

Think of:

Variable = house
Pointer = address of the house

An array is a data structure used to store multiple values of the same type in a single variable, organized in a sequence.

Simple Idea
Instead of creating many variables:
int a1 = 10, a2 = 20, a3 = 30;
You use one array:
int arr[3] = {10, 20, 30};
How It Works

Each value in an array has an index (position), starting from 0.

Index:   0    1    2
Value:  10   20   30
Example (C)
int arr[3] = {10, 20, 30};

printf("%d", arr[0]);  // Output: 10
printf("%d", arr[1]);  // Output: 20
Example (Python)
arr = [10, 20, 30]
print(arr[0])  # 10
Key Points
Stores multiple values in one variable
All elements are of the same data type (in languages like C, Java)
Accessed using index
Index starts from 0
Real-Life Example

Think of an array like a row of lockers:

Each locker has a number (index)
Each locker stores an item (value)
